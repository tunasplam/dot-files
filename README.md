# Dot Files

These is my new dot files repo. This is NOT going to be having any sort of install scripts, unless they are super custom.

## Usage

Clone the repo and then replace your dot files with sym links to the corresponding dot files in this repo.

## Font Instructions

[Download GeistMono Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/GeistMono.zip)

Why?
	- monospaced
	- looks nice
	- supports ligatures

[Install Instructions](https://wiki.debian.org/Fonts)
