# Dot Files

These is my new dot files repo. This is NOT going to be having any sort of install scripts, unless they are super custom.

## Usage

Clone the repo and then replace your dot files with sym links to the corresponding dot files in this repo.

## Prerequisites

- [zsh](https://wiki.debian.org/Zsh)
- [Starship](https://starship.rs/)
- [kitty terminal](https://wiki.archlinux.org/title/Kitty)
- [pokefetch](https://github.com/tunasplam/Pokefetch) (my own fork)
- [pywal](https://github.com/dylanaraps/pywal)
- [GeistMono Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/GeistMono.zip) ([Install Instructions])(https://wiki.debian.org/Fonts)
- [VS Code](https://wiki.debian.org/VisualStudioCode)
